# 🌐 Site da Ketlin Vith

Projeto de um site estilo **link na bio**, desenvolvido para centralizar e facilitar o acesso às redes sociais e conteúdos.

---

## Sobre o projeto

Este é um site simples e funcional criado para centralizar links importantes em um único lugar, facilitando o acesso para visitantes.

O projeto foi desenvolvido seguindo o conceito de **mobile-first**, priorizando a experiência em dispositivos móveis.

---

## Tecnologias utilizadas

* HTML
* CSS

---

## Funcionalidades

* Página de links centralizados
* Navegação simples e intuitiva
* Layout otimizado para celulares e tablets
